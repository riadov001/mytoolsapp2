# MyTools Admin — Product Hunt Description

## Short Description (tagline)
**The garage management app built for performance** 🔧

## One-liner
MyTools Admin brings your entire garage operation — quotes, invoices, appointments, and clients — to your pocket.

---

## Full Product Description

Running a garage means juggling a hundred things at once. Paper quotes get lost. Invoice tracking lives in spreadsheets. You miss appointments because the schedule's on a Post-it note.

**MyTools Admin changes that.**

We built a mobile-first dashboard for garage owners and administrators that connects directly to the MyTools Group back-office. Real data. Real time. No Excel.

### What you can do in 30 seconds

- ✅ **Check your revenue** — monthly and cumulative, right on the dashboard
- ✅ **Create a quote** — fill in the details, attach a photo, send it
- ✅ **Mark an invoice as paid** — one tap, done
- ✅ **Book a reservation** — calendar picker, client selection, service type
- ✅ **Look up a client** — search, view history, edit info

### Built differently

Most garage software is desktop-first, slow, and designed for accountants. MyTools Admin is built for the person standing in the garage, hands dirty, needing to pull up a quote in 5 seconds on their phone.

Dark mode by default. Fast. Clean. Secure.

### Enterprise-grade security

This isn't a public app. Every user is manually vetted and provisioned by MyTools Group. JWT authentication, encrypted sessions, zero tracking. Your client data stays yours.

### Who it's for

Partner garage administrators and employees in the MyTools Group network. Currently available in France.
